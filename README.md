# LeadPro

האתר עכשיו כולל מספר דפים סטטיים, עיצוב מרכזי וטעינה נכונה של הטופס המוביל.

קבצים מרכזיים:

- `index.html` - דף הנחיתה הראשי עם הטופס.
- `about.html` - דף אודות השירות.
- `services.html` - דף שירותים.
- `styles.css` - עיצוב משותף לכל הדפים.
- `script.js` - סקריפט EmailJS עבור הטופס.
- `assets/hero.svg` - גרפיקה לשימוש בעמוד הבית.

הטופס המקורי שתוקן נשמר בדיוק כפי שהיה ב-`script.js`, כולל קריאת EmailJS והסלקטורים של שדות הטופס.
